# Webshop End-to-End Test

## Purpose
Example of Playwright end-to-end test on the [EK Webshop](https://github.com/arturomorarioja/js_webshop) sample. The test is coded in JavaScript.

## Instructions
The webshop is expected to run at `http://127.0.0.1:5500/`. This value can be changed in `playwright.config.js`.

Run the tests: `npm run test`

Playwright / JavaScript
